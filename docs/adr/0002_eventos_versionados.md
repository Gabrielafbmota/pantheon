# ADR 0002 — Eventos versionados como integração primária

## Contexto
Integrações síncronas aumentam acoplamento e risco de cascata.

## Decisão
Eventos versionados serão o padrão de integração principal; APIs para leitura/comandos administrativos.

## Consequências
- Evolução mais segura
- Necessidade de catálogo e validação de schema
