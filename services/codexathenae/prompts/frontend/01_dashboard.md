# Frontend • Tela 01 — Dashboard

## Objetivo
Criar dashboard com KPIs do catálogo + insights básicos (quando existir).

## Entradas
- API: GET /books (total) + agregações futuras
- Layout: dark mode, sidebar, cards

## Saídas esperadas
- Componentes React + Zustand store
- Loading/empty/error states
- Testes (opcional) com vitest

## Restrições / Regras
- Consumir exclusivamente a API
- Responsivo (mobile/desktop)
- Acessibilidade básica

## Tarefas
1) Criar `useBooksSummary` hook
2) Layout cards
3) Skeleton loading
4) Tratamento de erro

## Critérios de pronto
- Dashboard renderiza com dados reais
- Não quebra sem insights
