# Frontend • Tela 05 — Reader PDF

## Objetivo
Implementar reader PDF com pdf.js com progresso (page+scroll).

## Entradas
- PDF via URL/file_id
- Eventos: page/scroll

## Saídas esperadas
- Persistência local + sync
- UI: zoom, page nav

## Restrições / Regras
- Throttle de scroll
- Performance em mobile

## Tarefas
1) Integrar pdf.js
2) Capturar page+scroll
3) SyncQueue
4) UI controls

## Critérios de pronto
- Retoma page/scroll
- Não consome CPU demais
