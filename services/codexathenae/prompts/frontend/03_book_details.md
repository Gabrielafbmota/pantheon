# Frontend • Tela 03 — Detalhes do Livro

## Objetivo
Mostrar detalhes completos e ações (editar/deletar).

## Entradas
- API: GET /books/:id
- Ações: PUT/DELETE

## Saídas esperadas
- Página com seções: metadados, descrição, tags
- Botões de ação
- Confirmação para delete

## Restrições / Regras
- Validar forms
- Proteger rotas (se auth ativado)

## Tarefas
1) Hook `useBookDetails`
2) Tela com layout
3) Modal delete
4) Form edit

## Critérios de pronto
- Operações funcionam via API
- Estados consistentes
