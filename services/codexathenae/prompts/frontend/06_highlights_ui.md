# Frontend • Tela 06 — Highlights/Notas

## Objetivo
Criar UI para criar/listar highlights ligados ao reader.

## Entradas
- API: POST /highlights, GET /highlights/:book_id

## Saídas esperadas
- Lista com busca local
- Criar nota
- Link para posição (quando possível)

## Restrições / Regras
- Sanitizar inputs
- UX simples

## Tarefas
1) Store highlights
2) Tela list
3) Form create
4) Integração com reader para locator

## Critérios de pronto
- Highlights persistem e aparecem corretamente
